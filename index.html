Welcome to ToolKit

ToolKit is an iPhone app that aims to be easily configurable by a researcher interested in environmental participatory sensing. ToolKit records environmental data using the phones sensors to be uploaded at the end of a session by a user. Each view is completely self contained and independent of the others.

If you have git installed, you can clone this project:

$ git clone git@github.com:eternalmatt/ToolKit.git

Or you can download a zip of the most recent version.

Sensors

Recent smartphones contain numerous sensors such as accelerometers, gps, cameras, microphones, and more. We aim to take advantage of these sensors in ToolKit in an elegant way such that a user can record data near them easily and upload to a researcher with minimal effort.

Self-Contained Views

Modularity is key. ToolKit provides many views that are completely self-contained and are unaware that other views exist. With this structure, we can allow a researcher to build an entire interface in minutes because they remain abstracted away from the implementation details of the sensors.
